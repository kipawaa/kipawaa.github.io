Mobius Function Calculator

Result:

This calculator determines the mobius function's value at n. The mobius function is often thought of as a square-detector. It outputs 0 if n is divisible by a prime squared. If a number is "square-free" (is not divisible by a prime squared) then the mobius function will return 1 if n has an even number of prime factors, and -1 if n has an odd number of prime factors.
